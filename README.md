# Siyana Board

A simple Kanban board built with vanilla JavaScript to organize tasks using drag-and-drop.

## Live Demo
https://siyana-code.github.io/project-03-siyana-board/

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## What I Practiced
- Storing and managing tasks with JavaScript arrays
- Saving and loading data using `localStorage`
- Using the HTML5 Drag & Drop API (`dragstart`, `dragover`, `drop`)
- Creating and updating DOM elements dynamically
- Iterating through arrays with `forEach()` to render tasks
- Keeping the UI and stored data synchronized

## Run Locally
1. Clone the repository.
2. Open `index.html` with Live Server (or any static server).
